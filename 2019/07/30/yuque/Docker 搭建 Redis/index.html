命令 含义
-d 本地运行
6379:6379 本地端口:Docker 端口
-v 指定驱动盘
G:\redisData Redis 的持久化文件存储
redis docker 的镜像名
redis-server redis 服务器
–appendonly yes 开启持久化
–name 执行这个运行的名称
–restart=always Docker 启动容器就启动
