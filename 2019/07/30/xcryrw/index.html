Docker 搭建 halo 博客

2019.07.30 glacier  热度 ℃

halo 博客是一款轻量级的 java 博客 https://github.com/halo-dev/halo

在/usr/ 目录下下载 halo的jar包

halo 默认的端口是 8090

打开浏览器：http://ip:8090 就可以访问了
