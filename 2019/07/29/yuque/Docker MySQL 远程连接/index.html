配置容器

下载 mysql 容器

将本地 33306 端口映射到容器 3306 端口

进入容器

设置远程登录

登录 mysql，密码就是我们刚刚设置的 MYSQL_ROOT_PASSWORD=123456

root 远程登录权限

退出 mysql，退出容器

Navicat 测试连接
